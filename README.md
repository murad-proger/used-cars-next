# Car Marketplace (Fullstack Next.js App)

Fullstack web application for managing and selling cars built with:

- Next.js (App Router)
- PostgreSQL (Docker local development)
- NextAuth authentication
- REST API routes
- Migration planned to Supabase
- Deployment target: Vercel

---

## Features

- User authentication (NextAuth)
- Admin panel for managing cars
- Add / edit / delete products
- Image support for listings
- Shopping cart & checkout system
- PostgreSQL database integration

---

## Tech Stack

- Next.js 14+
- TypeScript
- PostgreSQL (Docker)
- NextAuth
- Docker (local DB)
- Supabase (planned migration)
- Vercel (deployment)

---

## Getting Started

```bash
npm install
```
